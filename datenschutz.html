Schriftarten

Auf dieser Website werden Schriftarten lokal eingebunden und auf dem eigenen Server bereitgestellt. Eine Verbindung zu externen Font-Anbietern (z. B. Google Fonts) findet nicht statt.

Dadurch werden beim Aufruf der Website keine personenbezogenen Daten (insbesondere keine IP-Adressen) an Dritte im Zusammenhang mit Schriftarten übermittelt.
